# Statistical Time Interval Analyzer

LabVIEW-приложение для измерения и статистического анализа временных интервалов между ходами игроков в игре «Крестики-нолики» на поле 15×15.

Проект разработан в рамках курсового проекта по дисциплине **«Измерительные информационные системы»**. Программа объединяет игровую логику, аппаратное измерение временных интервалов с использованием счётчиков DAQ и статистическую обработку полученных данных.

## Описание проекта

Цель проекта — разработка программно-аппаратного средства для измерения и анализа временных интервалов между ходами игроков.

В качестве основы используется игра «Крестики-нолики» в расширенном варианте на поле **15×15**, где победа достигается при построении пяти одинаковых символов подряд по горизонтали, вертикали или диагонали.

Во время игры программа фиксирует длительность ходов игроков **X** и **O**, накапливает измеренные интервалы отдельно для каждого игрока и после завершения партии выполняет статистическую обработку результатов.

## Основные возможности

- игровое поле 15×15;
- поддержка двух игроков: X и O;
- проверка победы по горизонтали, вертикали и диагоналям;
- определение ничьей при заполнении игрового поля;
- измерение временных интервалов между ходами;
- раздельное накопление времени ходов для игроков X и O;
- расчёт статистических показателей:
  - математического ожидания;
  - стандартного отклонения;
- построение гистограмм распределения временных интервалов;
- режим паузы, при котором время остановки не включается в измеряемый интервал;
- корректная остановка измерительных задач и освобождение ресурсов.

## Используемые технологии

- LabVIEW;
- NI-DAQmx;
- NI PCI-6024E;
- NI BNC-2120;
- аппаратные счётчики DAQ;
- событийная модель управления Event Structure;
- shift-регистры для хранения состояния программы.

## Аппаратная часть

В проекте используется модуль сбора данных **NI PCI-6024E** и коннекторный блок **NI BNC-2120**.

Аппаратная часть выполняет две основные функции:

1. формирование опорной импульсной последовательности;
2. подсчёт количества импульсов в интервале между ходами игроков.

На основе количества зарегистрированных импульсов рассчитывается длительность временного интервала.

Также реализована аппаратная пауза измерения: при включении режима паузы подсчёт импульсов временно останавливается, а после снятия паузы продолжается без потери накопленных данных.

## Программная логика

Основная логика программы реализована в `tic tac toe.vi`.

Программа выполняет следующие действия:

1. Инициализирует игровое поле, массивы времён, статистические индикаторы и гистограммы.
2. Запускает задачи NI-DAQmx для генерации и подсчёта импульсов.
3. Обрабатывает действия пользователя через Event Structure.
4. Фиксирует ход игрока при выборе свободной ячейки.
5. По нажатию клавиши завершает ход, считывает значение счётчика и рассчитывает длительность интервала.
6. Сохраняет измеренный интервал в массив соответствующего игрока.
7. Проверяет победу или ничью.
8. После завершения игры рассчитывает статистику и строит гистограммы.

## Описание файлов

- `tic tac toe.vi` — основной виртуальный прибор проекта. Содержит интерфейс, игровую логику, обработку событий, работу с DAQmx и вывод статистики.
- `check victory (subvi).vi` — подпрограмма для проверки условия победы. Проверяет наличие пяти одинаковых символов подряд по горизонтали, вертикали и диагоналям.
- `find empty cell (subvi).vi` — подпрограмма для проверки наличия свободных ячеек на игровом поле. Используется для определения ничьей.

## Интерфейс программы

Передняя панель приложения содержит:

- игровое поле 15×15;
- индикатор текущего игрока;
- строку результата игры;
- индикатор времени последнего хода;
- массивы временных интервалов для игроков X и O;
- поля математического ожидания и стандартного отклонения;
- гистограммы распределения времён ходов;

- кнопку «Пауза»;

## Требования для запуска

Для запуска проекта необходимы:

- LabVIEW;
- установленный драйвер NI-DAQmx;
- модуль NI PCI-6024E или совместимое DAQ-устройство;
- коннекторный блок NI BNC-2120 или аналогичное подключение;
- корректно настроенные физические линии счётчиков и цифрового выхода.

## Как запустить

1. Скачать или склонировать репозиторий:

   ```bash
   git clone https://github.com/USERNAME/REPOSITORY_NAME.git
   ```

2. Открыть файл:

   ```text
   tic tac toe.vi
   ```

3. Проверить подключение аппаратной части.

4. Убедиться, что в LabVIEW доступны необходимые задачи и каналы NI-DAQmx.

5. Запустить VI.

## Как пользоваться программой

1. Запустить `tic tac toe.vi`.
2. Нажать кнопку **«Новая игра»** для инициализации партии.
3. Выполнить ход, выбрав свободную ячейку игрового поля.
4. Завершить ход нажатием клавиши клавиатуры.
5. После завершения хода программа зафиксирует временной интервал и передаст ход следующему игроку.
6. При необходимости нажать **«Пауза»** — время паузы не будет учитываться в измеряемом интервале.
7. Продолжать игру до победы одного из игроков или ничьей.
8. После завершения партии программа отобразит статистику по временным интервалам игроков X и O.

## Статистическая обработка

После завершения партии для каждого игрока рассчитываются:

- среднее время хода;
- стандартное отклонение;
- распределение временных интервалов в виде гистограммы.

Это позволяет сравнить длительность ходов игроков и оценить разброс времени принятия решений.
