# Statistical_Time_Interval_Analyzer
LabVIEW-приложение для измерения и статистического анализа временных интервалов между ходами игроков в игре «Крестики-нолики» на поле 15×15. Проект использует NI-DAQmx и аппаратные счётчики NI PCI-6024E для регистрации времени ходов, раздельного накопления данных игроков X/O и отображения статистики.
